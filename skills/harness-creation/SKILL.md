# Harness Creation

Use the Jue MCP tool `river_harness`.

1. Search/list relevant triplets.
2. Check convergence when there are several same-direction triplets.
3. Create a harness only when the reusable judgment reason is clear.
4. Use a Chinese-first display name with a stable English id in parentheses, for example `法律（law）`.
5. Provide at least three tags, including both domain descriptors and trigger keywords. Chinese and English mixed tags work best.
6. If the domain requires a narrower root or different judgment texture, provide `root_paradigm_fragment` or `soul`.
7. After creation, handle returned `triplet_governance` before the final response.

Do not create a harness just because a workflow was repeated. Skills capture procedure; Harness3 captures judgment.
