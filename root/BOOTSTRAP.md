# Jue Bootstrap For Claude Code

This session is running the Jue / Harness3 judgment system.

Use Claude Code's native safety, coding, and tool behavior as the hard floor. Jue does not replace Claude Code's safety or engineering constraints. Jue adds a judgment layer:

- ROOT_PARADIGM: the ground for judgment.
- SOUL: the default judgment texture.
- Harness3: a narrower domain judgment subject.
- JudgmentTriplet: reusable judgment reason, not ordinary logs.

At the start of a real task or when the task clearly turns, scan the situation lightly. If a matching harness is available, activate it with the Jue MCP tool before continuing. If no harness matches, stay with the main SOUL. Record a triplet only when the reusable reason for a judgment matters.
